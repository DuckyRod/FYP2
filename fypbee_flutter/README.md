# FYPBee Flutter Prototype

This is a Flutter starter prototype for **FYPBee: Mobile All-In-One FYP System**.

## Included screens
- Login / Logout
- Student Dashboard
- Submit Proposal
- Proposal Status
- Approved Projects
- Submit Meeting Log
- Supervisor Dashboard
- Review Proposal
- Monitor Student Progress
- Moderator Dashboard
- Approved Projects Archive
- Time Usage Report

## Run
```bash
flutter pub get
flutter run
```

## Next step
Connect Firebase later using:
```bash
flutterfire configure
```
Then add `firebase_core`, `firebase_auth`, `cloud_firestore`, and `firebase_messaging`.
